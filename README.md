A modern and interactive Loading Screen Animation built using HTML, CSS, and JavaScript. This project displays a dynamic progress bar with a loading percentage indicator, providing users with a smooth and engaging loading experience.

🚀 Features
Animated loading progress bar
Real-time percentage counter
Modern and clean user interface
Smooth loading animations
Responsive design
Lightweight and easy to customize
🛠 Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📖 Project Overview

This project simulates a loading process by gradually increasing a progress bar and updating the loading percentage. It can be used as a preloader or loading screen for websites, web applications, and dashboards to improve user experience.

🎯 Learning Objectives

This project helps developers learn:

JavaScript DOM Manipulation
Dynamic Content Updates
CSS Animations and Transitions
Progress Bar Implementation
Frontend UI Development
💡 Use Cases
Website preloaders
Application loading screens
Dashboard initialization screens
Interactive UI projects
Frontend development practice
📂 Project Structure
HTML – Page structure and loader elements
CSS – Styling and animation effects
JavaScript – Dynamic progress and percentage updates
